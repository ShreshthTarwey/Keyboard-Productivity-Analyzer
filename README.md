# Keyboard Activity & Productivity Analyzer

## Overview
A Python-based tool to track keyboard activity, analyze typing speed, identify errors, and monitor idle time. This tool helps users understand their productivity patterns.

## Features
- **Key Logging**: Tracks keystrokes securely in the background.
- **Productivity Analysis**: Calculates typing speed (WPM) and error rates.
- **Reporting**: Generates session reports.

## Technologies
- Python 3.x
- `pynput`
- `pandas`
